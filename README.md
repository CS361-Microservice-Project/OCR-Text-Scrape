# OCR-Text-Scrape
A microservice that takes in an image and returns scraped text.


#Installation Instructions
1. Open terminal in desired directory on your local machine.
2. Enter the following: git clone https://github.com/CS361-Microservice-Project/OCR-Text-Scrape.git
3. cd OCR-Text-Scrape

   (Windows)
5. python -m venv venv
6. venv\Scripts\activate

   (Mac/Linux)
5. python3 -m venv venv
6. source venv/bin/activate

7. pip install -r requirements.txt
8. python main.py
